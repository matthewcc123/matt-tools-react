# Matt's Tools App

A collection of simple utilities built to streamline everyday workflows and improve productivity.

## Tech Stack

- **Current:** React.js
- **Previous:** WinForms → .NET MAUI Blazor → WinUI3

## Project Background

This project started as a Windows Forms prototype, evolved into a .NET MAUI Blazor application, and has now been rebuilt with **React.js** for a simpler, faster, and more maintainable web-based experience.
